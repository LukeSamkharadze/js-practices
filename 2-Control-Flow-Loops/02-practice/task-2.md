####Task 2

Replace `if..else` to several ternary operators. 

For readability - write the code in several lines.

```js
var message;

if (login == 'Pitter') {
  message = 'Hi';
} else if (login == 'Owner') {
  message = 'Hello';
} else if (login == '') {
  message = 'unknown';
} else {
  message = '';
}
```
